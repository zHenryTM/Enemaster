# Enemaster 📚

Gerador de lista de questões do Enem, usando com filtro as Habilidades da [Matriz de Referência do Inep](https://download.inep.gov.br/download/enem/matriz_referencia.pdf). Assim, você pode identificar os padrões da prova e se preparar adequadamente.

## O que são as Habilidades ✅

Cada área do conhecimento (Linguagens, Humanas, Naturezas e Matemática) é composta por 30 habilidades. Cada questão da prova de uma dessas áreas é feita em cima de uma dessas 30 habilidades, algumas podendo se repetir, a fim de compor as 45 questões de cada matéria.

Elas são bem definidas na [Matriz de Referência do Inep](https://download.inep.gov.br/download/enem/matriz_referencia.pdf). Assim, ao praticar individualmente cada uma das habilidades, o estudante será capaz de reconhecer os padrões da prova, quais conteúdos priorizar e quais estudar menos, além de desenvolver resistência e estratégia de prova.

## Como estudar com o Enemaster 🔥

Após baixar sua lista de exercícios, resolva-a por completo e sem pausas. Use um tempo médio de 3 minutos para resolver cada questão para treinar sua agilidade. Enquanto estiver resolvendo a lista, pule as questões que você travar, as que estiver demorando, as que você não sabe responder e as que você sabe que consegue fazer, mas que levará um bom tempo. 

Desse modo, você fará as questões mais fáceis da lista primeiro, ganhando tempo para poder gastar com as questões mais difíceis/demoradas. Isso é fundamental.

Depois de resolver a lista, corrija as questões que você errou e as questões que você acertou, mas teve dúvida em alguma parte. Depois, crie flashcards dessas questões e as refaça espaçadamente (recomendo o uso do Anki para isso). Assim, você fechará as lacunas de seus conhecimentos.

## Como contribuir com o projeto 👨‍💻

Se você encontrar algum bug, ou quiser fazer alguma melhoria no Enemaster, sinta-se livre para mexer no código como quiser. Para tanto, siga os passos abaixo:

1. Certifique-se de ter o [Git](https://git-scm.com/) instalado em sua máquina
2. Faça um Fork no projeto
3. Crie um clone de seu Fork `git clone https://github.com/seu-usuario/nome-fork.git`
4. Crie uma nova branch `git checkout -b nome_branch_nova`
5. Instale as dependências no terminal com o comando `pip install requirements.txt`
6. Faça um commit `git add .` --> `git commit -m "minhas_alterações"`
7. Envie seu commit para seu Fork `git push origin minhas_alterações`
8. Abra um Pull Request (PR):
   - Vá em seu Fork no Github
   - Clique em "Compare & pull request"
   - Explique o que você fez, o porquê você fez e como testar suas alterações
